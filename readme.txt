== Description ==

Upload your custom logo for WordPress login page instead of the usual WordPress logo.

Kindly let us know your feedback or comments to add more features in this plugin.

== Installation ==

1.Log in to your WordPress admin panel and go to Plugins -> Add New
2.Type Custom Admin Login Logo in the search box and click on search button.
3.Find Custom Admin Login Logo plugin.
4.Then click on Install Now after that activate the plugin.

== Frequently Asked Questions ==

= How to configure Change WordPress Login Logo Plugin? =

Click “Settings” tab from left navigation menu:
Click “Wp Login Logo” menu under the Settings menu
Upload new login logo using WP media uploader.
Click “Save Changes” button to save the configuration details.
